# Honest Texas Electricity

A free, no-affiliate website that helps Texas consumers find the honest cheapest
electricity plan. Filters PowerToChoose data, exposes gimmick plans, ranks by true cost.
Independence (no referral money) is the product's whole value proposition.

See the design doc (`~/.gstack/projects/Untitled-Power-Project/`) for architecture and
the sequenced milestone plan. Start with Milestone 1: the Oncor data pipeline.
